# Envio de Facturas automatico

::alert{type="info"}

Esta es una guía para programar el envío de facturas a SUNAT.

::

## ¿Qué es el envío de Facturas automatico?

El envío de facturas automatico es una funcionalidad que permite enviar facturas a SUNAT de forma automática.

## ¿Cómo programar el envío de Facturas automatico?

1. Ir a **Configuración** -> **Configuración Globales** -> **Avanzado** -> **Tareas Programadas**

![alt text](img/envio-facturas-automatico-1.png)

2. Hacer clic en **Agregar**
3. Seleccionar **Para enviar facturas a SUNAT**

![alt text](img/envio-facturas-automatico-2.png)

4. Configurar los parámetros (Tarea y hora)
5. Hacer clic en **Guardar**

::alert{type="warning"}

**Nota:** Se recomienda programar el envío de facturas en horarios fuera de pico para evitar errores, se recomienda que sea en horarios de madrugada 1am a 5am.

::
